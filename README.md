# 🛠️ Modding Editor Pro (Auto-Save Edition)

**Modding Editor Pro** adalah editor berasaskan web yang direka khas untuk memudahkan para modder (terutamanya pemain Left 4 Dead 2) menguruskan CVAR, fail AddonInfo, dan skrip `.nut` (Squirrel) secara terus dari pelayar web.

![Status](https://img.shields.io/badge/Status-Pro-blueviolet?style=for-the-badge)
![Tech](https://img.shields.io/badge/Tech-HTML5_/_CSS3_/_JS-orange?style=for-the-badge)

## ✨ Ciri-Ciri Utama
* **Triple Editor Support:** Tiga slot berbeza untuk CVAR, AddonInfo, dan Scripting.
* **Auto-Save System:** Data tidak akan hilang! Semua teks disimpan secara automatik dalam `localStorage` pelayar anda.
* **Google AI UI Experience:** Antaramuka yang responsif dengan kesan visual *Shimmer*, *Rotating Borders*, dan *Progress Bar* bertema Google AI.
* **One-Click Download:** Tukar kod anda kepada fail `.nut`, `.cfg`, atau `.txt` dengan sekali klik.
* **Drag & Drop:** Boleh tarik fail dari PC terus ke dalam kotak editor untuk edit dengan pantas.
* **Mobile Optimized:** Sesuai digunakan pada telefon pintar untuk rujukan atau suntingan pantas.

## 🚀 Cara Penggunaan
1.  **Input:** Tulis atau *paste* kod modding anda ke dalam salah satu kotak editor.
2.  **Edit:** Gunakan pintasan papan kekunci (cth: `Space` atau `Enter`) dan biarkan sistem *auto-save* bekerja.
3.  **Export:** Pilih format fail yang anda mahukan (cth: `.nut` untuk skrip L4D2 atau `.cfg` untuk autoexec) dan tekan butang muat turun.

## ⌨️ Pintasan Papan Kekunci (Shortcuts)
* **Ctrl + F:** Cari perkataan spesifik dalam kod.
* **Space:** Play/Pause (Jika diintegrasikan dengan pemain media).
* **Auto-Format:** Sistem akan membetulkan indentasi secara automatik apabila anda melakukan *paste*.

## 📂 Struktur Fail Utama
* `mod.nut` / `script.nut` - Fail skrip utama untuk Source Engine.
* `autoexec.cfg` - Fail konfigurasi untuk tetapan CVAR permainan.
* `addoninfo.txt` - Fail maklumat penting untuk setiap mod VPK.
